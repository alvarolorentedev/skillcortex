# Slm Cortex v0.1.1 Release Notes

Slm Cortex v0.1.1 is a patch release on top of the first public product cut.

## Summary

This release keeps the existing runtime and packaging behavior intact while
making the Composer app easier to invoke from an installed package and aligning
public version metadata with that entrypoint update.

## Highlights

- New `slmcortex-composer` console script for direct Composer app launch
- Package metadata and default packaged artifact versions bumped to 0.1.1
- Current public release notes linked from the GitHub issue template

## What changed

- Installed packages now expose both `slmcortex` and `slmcortex-composer`
- Default CLI artifact version flags now emit 0.1.1 instead of 0.1.0
- Release documentation now includes a dedicated note for this patch release

## Limitations

- Runtime capabilities, adapter formats, and composition behavior are unchanged
- Demo inference and agent execution still rely on dry-run validation of
  routing and control flow
- The compatibility server remains intentionally minimal and non-streaming
